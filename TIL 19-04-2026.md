# TIL 19-04-2026

Today I learned about the Python scripts in this project, which demonstrate the Rich library for enhanced console output.

## debug_example_with_richer.py

This script showcases Rich's debugging and error handling capabilities. It installs Rich's traceback handler, logs function calls with locals, performs some additions in a loop, intentionally triggers a TypeError, catches and prints the exception beautifully, and saves the entire console output to an HTML file for later viewing.

## present_table.py

This script creates and displays a formatted table using Rich's Table class. It sets up columns for movie release dates, titles, and box office earnings, populates it with Star Wars movie data, and prints the table to the console with styling.

## richer_examples.py

This is a comprehensive example of Rich's console features. It defines a custom theme for success, error, and warning messages, demonstrates styled text with colors, bold, underline, emojis, prints data structures, and uses console logging with timestamps in a loop with sleeps.

## show_markdown.py

This script demonstrates Rich's markdown rendering capabilities. It defines two markdown strings with headers, paragraphs, and lists, converts them to Rich Markdown objects, and displays them side-by-side in columns within panels.
